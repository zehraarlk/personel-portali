# Admin paneli

`.\baslat.ps1` ile birlikte ayağa kalkar.

| Adres | Ne |
|--------|-----|
| http://127.0.0.1:5173/ | Portal anasayfa |
| http://127.0.0.1:5173/test | Test sayfası |
| http://127.0.0.1:5173/admin/ | Bu admin paneli (proxy → :5174) |

Tek başına:

```powershell
cd admin
npm install
npm run dev
```
